This application uses the Kerio APIs Client Library for PHP[1] distributed under SDK lincense[2].

Install instructions
--------------------
Download and extract it to this 'src_Kerio' directory.

The structure may look like this:

operator-ext-mngt/
└── src_Kerio/
    └── class/
    │   └── KerioApiException.php
    │   └── KerioApiInterface.php
    │   └── KerioApi.php
    │   └── KerioApiSocketInterface.php
    │   └── KerioApiSocket.php
    └── KerioOperatorApi.php
